# 中文期刊学术论文写作风格指南

## 总体风格

中文核心/普刊论文的写作风格介于学术严谨与可读性之间。核心原则：

- **简洁直接**：每句话都有信息量，不说废话
- **逻辑清晰**：段与段之间有明确的逻辑链条
- **客观理性**：陈述事实和结果，避免主观评价
- **术语统一**：同一概念全文使用同一术语

## 各部分的写作要点

### 摘要

- 第三人称视角（"本文""本研究"），不用"我们"
- 不引用参考文献、不使用缩写
- 数据说话："准确率提升了3.2个百分点"优于"取得了显著提升"
- 避免"首次""开创性""重大突破"——让审稿人自己判断

### 引言

- 漏斗式结构：大领域 → 具体问题 → 现有不足 → 本文工作
- "已有方法的不足"要具体：不是"效果不好"，而是"在XX场景下F1值低于70%，难以满足实际需求"
- 贡献列表要可验证：每条贡献对应一个具体的方法创新或实验结论
- 避免过度铺垫——引言不是教科书综述

### 相关工作

- **按技术维度分组**，不要按论文逐条罗列
- 每组 3-5 篇文献，说明共同思路和各自的局限
- 末尾 1-2 句总结不足，自然过渡到"本文方法"
- 批评要客观：说"该方法在XX方面存在局限"，不说"该方法很差"

### 方法

- 先给整体框架（一段话 + 框架图引用），再展开细节
- 公式后必须有文字解释其含义
- 变量命名要规范：标量小写斜体（$n$, $d$），向量粗体小写（$\mathbf{x}$），矩阵粗体大写（$\mathbf{W}$）
- 避免只贴公式不解释——审稿人最讨厌这种

### 实验

- 表格"本文方法"用加粗突出最优结果
- 分析要深入：不仅说"我们的方法在XX指标上提升了X%"，还要分析**原因**
- 如果某个指标不如基线，坦诚说明原因（比编造理由更学术）
- 消融实验要与方法模块一一对应

### 结论

- 不要简单复制摘要（最常见的错误）
- 回顾工作 → 总结发现 → 局限性 → 未来方向
- 局限性体现学术诚实，审稿人反而会加分

## 常见问题

### 语言问题

| 错误 | 正确 |
|------|------|
| 提出了一种新的方法 | 提出了一种XX方法 |
| 取得了很好的效果 | 在XX数据集上F1值达到XX% |
| 实验结果表明 | 在XX任务上的实验结果显示 |
| 随着深度学习的快速发展 | 深度学习在XX领域已广泛应用 |
| 引起了广泛关注 | 已成为XX领域的研究热点 |

### 格式问题

- "如表1所示"而非"如下表所示"（引用具体编号）
- 公式独立成行，编号右对齐
- 图表标题：图在下，表在上
- 参考文献引用用上标 [1] 或 (Author, Year)

### 学术规范

- 不自我吹捧
- 不贬低他人工作
- 区分"本文提出"和"已有方法"的边界
- 实验结果如实报告
