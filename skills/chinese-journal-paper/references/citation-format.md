# GB/T 7714-2015 参考文献著录规则详解

## 基本原则

GB/T 7714 是中国学术期刊的标准参考文献格式。核心要点：
- 作者不超过3人时全部列出，超过3人列前3人后加"等"（英文用"et al"）
- 作者姓名：中文正常写，英文"姓, 名首字母."
- 每条文献以"."结尾

## 期刊论文 [J]

```
[序号] 作者1, 作者2, 作者3. 论文标题[J]. 期刊名, 年, 卷(期): 起止页码.
```

中文示例：
```
[1] 张三, 李四, 王五. 基于注意力机制的文本分类方法[J]. 计算机学报, 2024, 45(3): 521-535.
[2] 刘一. 面向低资源场景的命名实体识别[J]. 中文信息学报, 2023, 37(8): 45-56.
```

英文示例：
```
[3] Vaswani A, Shazeer N, Parmar N. Attention is all you need[C]//Advances in Neural Information Processing Systems. 2017: 5998-6008.
[4] Devlin J, Chang M W, Lee K. BERT: Pre-training of deep bidirectional transformers for language understanding[C]//Proceedings of NAACL-HLT. 2019: 4171-4186.
```

## 会议论文 [C]

```
[序号] 作者1, 作者2. 论文标题[C]//会议论文集名. 城市: 出版者, 年: 起止页码.
```

```
[5] 陈一, 赵二. 融合知识图谱的推荐方法[C]//中国中文信息学会学术年会论文集. 北京: 清华大学出版社, 2024: 112-120.
```

## 专著 [M]

```
[序号] 作者. 书名[M]. 出版地: 出版者, 年.
```

```
[6] 周志华. 机器学习[M]. 北京: 清华大学出版社, 2016.
[7] Goodfellow I, Bengio Y, Courville A. Deep learning[M]. Cambridge: MIT Press, 2016.
```

## 学位论文 [D]

```
[序号] 作者. 论文标题[D]. 城市: 学校名, 年.
```

```
[8] 张三. 基于深度学习的情感分析方法研究[D]. 哈尔滨: 哈尔滨工业大学, 2023.
```

## 电子资源 [EB/OL]

```
[序号] 作者. 标题[EB/OL]. (发布日期)[引用日期]. URL.
```

```
[9] OpenAI. GPT-4 technical report[EB/OL]. (2023-03-15)[2024-01-10]. https://arxiv.org/abs/2303.08774.
```

## arXiv 预印本

arXiv 论文优先按会议/期刊引用（如果已正式发表）。未正式发表的：

```
[10] Author A, Author B. Title[J/OL]. arXiv preprint arXiv:XXXX.XXXXX, 2024.
```

或作为电子资源引用。

## 常见错误

| 错误 | 正确 |
|------|------|
| 作者名全部大写 | 姓全大写或首字母大写，名用首字母 |
| 多于3个作者全列出 | 列前3人 + "等"/"et al." |
| 缺少文献类型标识 [J]/[C]/[M] | 必须标注 |
| 卷期格式错误 | "卷(期)"不是"期.卷" |
| 页码用"-"连接 | 用起止页码，如"521-535" |

## 引用顺序

- 按**正文引用顺序**编号（不是按作者姓氏排序）
- 正文中用上标 [1] 或 (张三, 2024) 引用
- 同一处引用多篇文献：[1-3] 或 [1,3,5]
