# Academic Writing - OpenClaw Skill

Academic writing assistant for papers, theses, and publications

## Installation

### Via ClawHub
```bash
clawhub install academic-writing
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/academic-writing.git
cp -r academic-writing ~/.openclaw/skills/academic-writing
```
