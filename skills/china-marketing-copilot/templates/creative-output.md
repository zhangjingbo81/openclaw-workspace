# 创意输出模板

每个创意必须包含以下完整结构：

```
创意 {N}: {创意标题}

核心idea:
{一句话描述，15字以内}

传播形式: {微博文案/视频脚本/小红书图文/互动话题/跨界联动/直播活动}
目标受众: {具体人群}
平台适配: {微博/B站/小红书/抖音/公众号}

创意详情:
{完整创意描述，200-500字，包含执行细节}

传播逻辑:
为什么这个创意能打动目标受众: {分析}
预期传播路径: {描述用户如何参与和传播}
话题标签: #{主标签} #{辅标签1} #{辅标签2}

数据支撑:
类似案例: {来自知识库的参考案例}
受众洞察依据: {来自哪个数据文件}
技术事实核查: {产品卖点是否准确，标注 ✅/⚠️/🔴}

查重结果:
✅ 未发现相似已用创意 / ⚠️ 与 {XX} 有相似性，建议调整

风险评估:
按 risk-assessment.md 模板执行快速检查，不在此处硬编码检查维度。
重点关注：技术事实准确性、行业黑话避雷（查 ecosystem/industry-memes.md）。

预期评论区反应（必填，按 comment-personas.md 的5类人群模拟）:
- 参数驱动型可能说: "{模拟评论}"
- 品牌忠诚型可能说: "{模拟评论}"
- 解构找茬型可能说: "{模拟评论}" ← ⚠️ 如果这条太尖锐，考虑调整创意
- 普通用户型可能说: "{模拟评论}"
```

## 各传播形式的要点速查

> 详细的平台传播规律见 [industry-ecosystem.md](../references/industry-ecosystem.md)

### 微博
- 短文案 + 争议点 + 话题标签
- 传播路径：大V首发 → 中小V跟进 → 素人铺量 → 热搜
- ⚠️ 评论区翻车风险最高

### B站
- 深度评测/技术科普/硬核拆解
- 传播路径：头部UP主 → 弹幕互动 → 二创
- ⚠️ 硬广会被喷，恰饭视频容易翻车

### 小红书
- 种草图文 + 真实体验 + 生活场景
- 传播路径：腰部博主 → 评论区互动 → 收藏转发 → 搜索沉淀
- ⚠️ 恰饭感明显 = 无效

### 抖音
- 3秒钩子 + 反转/对比/挑战
- 传播路径：爆款视频 → 算法推荐 → 模仿跟拍
- ⚠️ 过于硬核没人看

### 公众号
- 深度长文 + 有数据 + 有观点
- 传播路径：朋友圈转发 → 行业群 → 二次传播
- ⚠️ 标题党反噬
