# 新品类破局模板

## 使用说明

当产品属于市场认知尚不清晰的品类时，传统卖点传播无效——受众连"这是什么"都不清楚。

本模板提供通用流程框架。SKILL.md 中内置了5大破局方法论（认知刷新/感知锚定/尝鲜者探索/先锋创作者/专业信任纪录片），可直接调用。

---

## 通用流程

### Step 1: 诊断品类认知状态

```
品类诊断

当前认知状态:
□ 无人知晓 — 用户根本不知道这个品类存在
□ 有误解 — 用户知道但理解错了
□ 有竞品定义了品类 — 市场已有龙头
□ 认知模糊 — 听说过但不清楚价值

目标认知: {你希望受众怎样理解这个品类}
认知差距: {当前状态和目标之间的差距}
传播挑战: {最大的传播障碍是什么}
核心卡点: {市场教育成本高 / 价格门槛高 / 场景不清 / 内容同质化 / 缺乏背书}
```

### Step 2: 选择破局方法

基于核心卡点，从以下方法论中选择：

| 核心卡点 | 推荐方法 | 一句话说明 |
|---------|---------|-----------|
| 市场教育成本高 | 认知刷新法 | 找到已知标准，用产品刷新它 |
| 价格门槛高 | 感知价值锚定法 | 用超高价锚定，让主力价显得值 |
| 使用场景不清 | 尝鲜者探索法 | 让用户告诉你该怎么用 |
| 内容同质化 | 先锋创作者法 | 找先锋探索可能性，不是证明产品好 |
| 缺乏专业背书 | 专业信任纪录片法 | 让专业用户长期实测 |

也可以组合使用（详见 SKILL.md 的"新品类破局"章节）。

### Step 3: 设计传播方案

基于选定的方法，设计具体传播方案：

```
传播方案

目标人群: {首批触达谁}
传播形式: {什么载体}
核心信息: {说什么}
传播路径: {怎么扩散}
衡量指标: {怎么判断有效}
预算分配: {各环节预算}
```

### Step 4: 风险评估

新品类传播的风险评估尤其重要：
- 品类认知不清 = 传播信息更容易被误读
- 没有参考系 = 用户容易拿相近品类类比（类比可能不对）
- 少数负面可能定义整个品类印象

按 [risk-assessment.md](risk-assessment.md) 模板执行评估。

---

## 输出格式

```
新品类破局方案

品类诊断
{Step 1 内容}

破局方法
{选择的方法 + 选择理由}

传播方案
{Step 3 内容}

风险评估
{Step 4 内容}

诚实声明
如果当前知识库数据不足以支撑破局判断，在此说明。
"暂未找到明确破局点"比编造方案更有价值。
```
