# 3C行业传播生态框架

## 平台生态

### 微博 — 话题发酵场
- 核心机制：热搜 + KOL组合拳
- 传播节奏：大V首发 → 中小V跟进 → 素人铺量 → 热搜 → 媒体跟进
- 有效内容：争议性话题、数据对比、名人互动
- 风险点：评论区翻车率高、热搜成本高、容易引战
- 典型KOL：数码闲聊站、冰宇宙（爆料向）；钟文泽、WHYLAB（评测向）

### B站 — 深度评测阵地
- 核心机制：长视频 + 弹幕文化 + 二创
- 传播节奏：头部UP主评测 → 弹幕互动 → 社区讨论 → 二创扩散
- 有效内容：深度技术分析、真实使用体验、硬核拆解
- 风险点：弹幕不可控、硬广被喷、恰饭视频容易翻车
- 典型KOL：极客湾、小白测评、爱否科技、笔吧评测室

### 小红书 — 种草转化场
- 核心机制：图文 + 真实感 + 搜索入口
- 传播节奏：腰部博主种草 → 评论区互动 → 收藏转发 → 搜索沉淀
- 有效内容：真实体验、生活场景、开箱晒单
- 风险点：恰饭感明显=无效、图文质量要求高
- 适合品类：耳机、穿戴、智能家居（主观体验型）

### 抖音 — 情绪放大器
- 核心机制：短视频 + 算法推荐 + 模仿跟拍
- 传播节奏：爆款视频 → 算法推荐 → 模仿跟拍 → 话题发酵
- 有效内容：3秒钩子、反转/对比/挑战、情绪共鸣
- 风险点：过于硬核没人看、容易流于表面、算法不可控
- 适合品类：手机、耳机（视觉冲击型）

### 公众号 — 深度阅读场景
- 核心机制：长文 + 朋友圈转发 + 行业群传播
- 传播节奏：深度文章 → 朋友圈 → 行业群 → 二次传播
- 有效内容：行业分析、深度评测、观点文章
- 风险点：标题党反噬、阅读量依赖标题
- 适合品类：笔记本、智能家居（信息深度需求高）

## KOL生态特征

各品类的KOL生态有显著差异。共性规律：

- 评测门槛越高的品类，头部KOL影响力越大（笔记本 > 手机 > 耳机）
- "恰饭"视频的观众容忍度因品类而异（手机最高，耳机/笔记本较低）
- 每个品类有独立的KOL圈子，不能跨品类通用
- 季度发布节奏的品类有明确的评测窗口期

各品类的KOL名单和风格特征，详见 [ecosystem/kols.md](../ecosystem/kols.md)

## 传播节奏规律

### 发售期（T-14 到 T+14）
- T-14: 预热开始，KOL收到评测机
- T-7: 解禁期，评测视频集中发布
- T-0: 发布会/开售
- T+7: 第一波用户反馈
- T+14: 传播窗口期结束，话题自然衰减

### 品类节奏差异

| 品类 | 发布节奏 | 最佳合作窗口 | 决策周期 |
|------|---------|-------------|---------|
| 手机 | 季度集中发布 | T-14（送测）+ T-7（解禁）+ T+3（首销） | 1-4周 |
| 笔记本 | 跟随CPU/GPU换代 | 年初（CES后）+ 年中（618前） | 2-8周 |
| 耳机 | 全年持续 | 电商大促前1个月（618/双11） | 1-3周 |
| 穿戴 | 全年持续 | 新品发布后1-2周 + 大促前 | 1-3周 |
| 智能家居 | 全年持续 | 装修季（春秋）+ 大促前 | 1-6个月 |
