# Rewrite checklist

Use this checklist before returning rewritten text.

- [ ] Meaning is preserved
- [ ] Tone matches the user’s intent
- [ ] Formulaic transitions were reduced where needed
- [ ] Promotional or inflated language was removed
- [ ] Repetition was reduced
- [ ] Sentence rhythm feels more natural
- [ ] Vague attributions were tightened or removed
- [ ] Chatbot residue was removed
- [ ] No false facts were introduced
- [ ] No private text was sent to any verifier
- [ ] Verification, if used, covers metrics only
